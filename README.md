# Tourist Center CRM

Система управления туристическим центром на базе Laravel и React.

## Технологии
- **Backend**: Laravel 11+
- **Frontend**: React 19, Vite, SCSS (с поддержкой тем и Glassmorphism)
- **Database**: PostgreSQL

## Функционал
- **Публичная страница**: Список услуг с возможностью подать заявку.
- **Гостевые заявки**: Возможность оставить заявку без регистрации.
- **Личный кабинет**: Управление заявками и профилем для пользователей.
- **Панель администратора**: Полная статистика, управление пользователями, услугами и всеми заявками системы.
- **Дизайн**: Современный интерфейс с использованием Glassmorphism, анимаций и адаптивной верстки.

## Быстрый старт

### 1. Установка зависимостей
```bash
composer install
npm install
```

### 2. Настройка базы данных
Создайте файл `.env`, настройте подключение к БД и выполните миграции:
```bash
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
```

### 3. Основные команды запуска
Для удобства разработки настроен одновременный запуск фронтенда и бекенда:

- **Запуск всего проекта (Vite + Laravel Serve):**
  ```bash
  npm start
  ```
- **Только фронтенд (Vite):**
  ```bash
  npm run dev
  ```
- **Только бекенд (Artisan serve):**
  ```bash
  npm run dev:backend
  ```

## Полезные команды
- **Сброс и повторное заполнение БД:**
  ```bash
  php artisan migrate:fresh --seed
  ```
- **Быстрое заполнение тестовыми данными для CRM:**
  ```bash
  php artisan app:seed-demo
  ```
- **Полный пересозданный набор тестовых данных с миграциями:**
  ```bash
  php artisan app:seed-demo --fresh
  ```
- **Кастомное количество записей:**
  ```bash
  php artisan app:seed-demo --users=30 --leads=50 --reviews=10 --categories=10 --services=30
  ```
- **Сборка для продакшена:**
  ```bash
  npm run build
  ```
